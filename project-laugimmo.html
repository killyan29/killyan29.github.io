Location de maisons

L’application est un site web de location de maisons inspiré du modèle Airbnb. Elle permet aux utilisateurs de consulter des annonces de logements, de créer un compte pour publier leurs propres annonces et de réserver des maisons proposées par d’autres utilisateurs. Le site fonctionne comme une plateforme de mise en relation entre propriétaires et locataires. L’accès à la consultation des annonces est libre, mais certaines fonctionnalités nécessitent une authentification.

Technologies

HTML5 CSS3 PHP MVC MySQL

Fonctionnalités

  • Affichage de toutes les annonces sans obligation de connexion
  • Authentification utilisateur (inscription, connexion, déconnexion)
  • Gestion des annonces (utilisateur connecté) : publication, modification, suppression d'annonces
  • Navigation fluide et animations CSS soignées.
